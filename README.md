# kdrama-twin-cv
Find your Korean drama twin! The project finds your drama twin using Computer vision algorithms and simple cosine similarity calculation to find the best matches

By running data_collect.py you can collect actor`s images from web sources (10 images per actor)

Then running preprocessing.py you can preprocess your collected dataset of images 

Afterwards , goes feature_axtraction using CV techniques ORB/SIFT 

Using the uploaded image or webcam image we can find the Krdama twin within your collected dataset of people , this will be done by matching methods

Finally you will get your twin!
